
# CodeKids Enterprise · 正式版部署

## 启动
docker-compose up -d

## 服务端口
- 前端: http://localhost:8080
- Java 后端: http://localhost:8080/api/
- Python 执行服务: http://localhost:8080/py/
- AI 服务: http://localhost:8080/ai/
- Redis: 6379
- RabbitMQ: 5672 (管理界面: 15672)
- MongoDB: 27017
- MySQL: 3307 （root/root1234）
- MinIO: 9000 (API), 9001 (控制台 admin/admin123)
